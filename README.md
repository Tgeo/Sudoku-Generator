Generates a valid sudoku board:

Execute sudoku-generator.py and it will be printed to stdout.

Example board generation:

$ ./sudoku-generator.py

[8, 1, 7, 9, 4, 5, 2, 6, 3]
[4, 6, 3, 8, 2, 1, 5, 7, 9]
[5, 2, 9, 3, 7, 6, 4, 1, 8]
[1, 4, 6, 7, 8, 9, 3, 2, 5]
[9, 3, 2, 6, 5, 4, 7, 8, 1]
